## 🚀 Sobre el Proyecto

Esta es la página de presentación (Landing Page) para **Pan Comido**, un moderno sistema integral de gestión para el sector gastronómico. El sitio está diseñado para presentar las características clave del producto, atraer prospectos (leads) y ofrecer demostraciones interactivas del sistema.

El diseño se enfoca en una estética limpia, moderna y profesional, utilizando técnicas de **Soft UI** y **Glassmorphism** sobre una paleta de colores claros, logrando una interfaz amigable que transmite simplicidad tecnológica.

## ✨ Características Principales

* **Menú QR:** Pedidos directos desde el celular del cliente.
* **Pago Móvil:** Múltiples medios de pago sin demoras.
* **Stock en Tiempo Real:** Control preciso de ingredientes y disponibilidad.
* **Sincronización:** Conexión sin latencia entre el salón, la cocina y la caja.

## 🛠️ Tecnologías Utilizadas

* **HTML5 Semántico:** Estructura accesible y optimizada para SEO.
* **CSS3 (Vanilla):** Implementación de diseño responsivo (Mobile First), variables CSS (Design Tokens), Flexbox y CSS Grid.
* **JavaScript (Vanilla):** Interactividad, animaciones de *scroll reveal* y modales sin dependencias de frameworks externos.
* **Diseño UI/UX:** Glassmorphism, degradados suaves y diseño responsivo fluido.

## 💻 Desarrollo Local

Para correr este proyecto en tu entorno local, no necesitas instalar dependencias complejas. Solo necesitas un servidor HTTP básico.

1. Clona este repositorio:
   ```bash
   git clone https://github.com/schamorro881/pan-comido-landing.git
   ```
2. Navega al directorio del proyecto:
   ```bash
   cd pan-comido-landing
   ```
3. Inicia un servidor local (por ejemplo, con Python):
   ```bash
   python3 -m http.server 8080
   ```
4. Abre tu navegador en `http://localhost:8080`
